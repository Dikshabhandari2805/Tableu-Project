# Trends in the Automotive Industry by Diksha Bhandari

An interactive **Tableau dashboard** designed to analyze trends in the **used car market**, focusing on pricing, fuel type, ownership patterns, transmission, and seller categories. This project transforms raw automotive data into meaningful business insights through visual storytelling.

This dashboard answers key questions such as:

* How do kilometers driven vary by fuel and transmission type?

The dashboard is built to be **interactive, intuitive, and insight-driven**, making it useful for analysts, recruiters, and business stakeholders.
